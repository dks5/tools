Инструкция по активации Windows через командную строку

Шаг 1: Запустите командную строку от имени администратора.

Для этого:

  • Нажмите правой кнопкой мыши на кнопку "Пуск".
  • Выберите "Windows PowerShell (администратор)" или "Командная строка (администратор)".
  • Если появится запрос контроля учетных записей пользователей, нажмите "Да".

Шаг 2: Введите поочередно следующие команды. После ввода каждой команды нажимайте клавишу Enter.

slmgr /ipk W269N-WFGWX-YVC9B-4J6C9-T83GX
slmgr /skms kms.digiboy.ir
slmgr /ato

Каждая команда выполняет определенное действие:

  • slmgr /ipk [ключ продукта] - устанавливает ключ продукта.
  • slmgr /skms [KMS-сервер] - устанавливает имя или IP-адрес KMS-сервера.
  • slmgr /ato - активирует Windows, обращаясь к KMS-серверу.

Шаг 3: Дождитесь сообщения об успешной активации Windows.

После выполнения команды slmgr /ato вы должны увидеть сообщение, подтверждающее успешную активацию.

Важно: Для успешной активации ваш компьютер должен быть подключен к интернету, чтобы иметь возможность связаться с указанным KMS-сервером.
